# 편의점 나이트 서바이벌 — MVP v1 Data Pack

This folder contains a **data-driven MVP** dataset for:
- 20:00 runs
- Infinite map
- Weapons 6 / Passives 6
- Evolutions via chest

## Files
- `weapons.json` (8 base weapons + 2 evolved)
- `passives.json` (8 passives)
- `evolutions.json`
- `enemies.json` (6 enemies + 1 elite + 1 boss)
- `stages.json` (1 stage)
- `spawn_tables.json` (time segments + threat curve)
- `upgrades_meta.json` (meta progression)

## Notes
- Numbers are tuned for a first playable balancing pass.
- Expect iteration after first playtest (TTK, spawn density, weapon dominance).
